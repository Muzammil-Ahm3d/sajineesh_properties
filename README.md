# Infrastructure Builders Hub

## Project info

This project is a React application built with Vite, shadcn-ui, and Tailwind CSS.

## Getting Started

To work locally:

1.  **Clone the repository**
    ```sh
    git clone <YOUR_GIT_URL>
    ```

2.  **Navigate to the project directory**
    ```sh
    cd infrastructure-builders-hub
    ```

3.  **Install dependencies**
    ```sh
    npm i
    ```

4.  **Start development server**
    ```sh
    npm run dev
    ```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
